# 论证顺序

> 规划全文的论证推进路线。写作时按此顺序展开，避免论证跳跃或重复。

## 总体论证策略

（选择一种或组合使用）

- [ ] **演绎式**：先给总论点，再分论点逐一论证
- [ ] **归纳式**：先呈现事实与案例，再归纳结论
- [ ] **问题驱动**：提出问题 → 分析原因 → 给出回答
- [ ] **对比式**：对立观点并列呈现 → 分析优劣 → 得出判断
- [ ] **时序式**：按事件发展顺序推进
