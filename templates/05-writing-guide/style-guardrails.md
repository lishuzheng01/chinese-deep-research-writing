# 写作风格护栏

> 本文件为当前任务定制的风格约束。在写作阶段与终稿质检阶段反复参照。

## 通用禁止项（适用于所有体裁）

- [ ] 不使用套话起手（"随着…的发展"等）
- [ ] 不使用空洞拔高（无证据的大词结论）
- [ ] 不使用三段式机械排比
- [ ] 不使用"首先/其次/最后"作为唯一推进手段
- [ ] 不以模糊主语替代真实来源（"有学者指出"但无出处）
- [ ] 不使用同义改写堆字数
- [ ] 不在结论中覆盖超出证据支持的范围

## 体裁特定约束

### 学术/论文体裁追加
- [ ] 保持克制、正式语气
- [ ] 避免第一人称（除方法说明部分）
- [ ] 术语首次出现时给出定义或出处

### 报告体裁追加
- [ ] 事实与建议严格分层
- [ ] 风险与权衡不可省略
- [ ] 多用表格、数据对比辅助阅读

### 特稿/深度文章追加
- [ ] 可使用场景、对比、叙事技巧
- [ ] 避免煽情与无据判断
- [ ] 兼顾节奏感与信息密度

### 新闻/解释性报道追加
- [ ] 报道与评论严格分离
- [ ] 使用精确、中性语言
- [ ] 每项事实陈述须有归因
