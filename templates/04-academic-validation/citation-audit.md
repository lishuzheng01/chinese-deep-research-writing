| 检查项 | 状态 | 问题描述 |

| 正文引用编号与参考文献列表一一对应 | 通过/不通过 | |
| 参考文献条目包含充分元数据 | 通过/不通过 | |
| 无虚构的作者、年份、页码或机构 | 通过/不通过 | |
| 无未标注来源的核心事实陈述 | 通过/不通过 | |
| 无"孤立"参考文献（列表中有但正文未引用） | 通过/不通过 | |
| 同一来源编号全篇一致（含分章文件） | 通过/不通过 | |
| 引用格式统一遵循 GB/T 7714 | 通过/不通过 | |
| 网络来源标注了引用日期 | 通过/不通过 | |
