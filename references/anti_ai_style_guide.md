# 去 AI 痕迹写作规范

> 本文件定义写作过程中必须遵守的去 AI 痕迹规则。在 Step 4（写作）和 Step 5（质检）阶段加载参考。

## 必须避免

- 套话起手（如"随着时代发展"、"在当今社会"、"众所周知"）
- 空洞拔高（无证据的大词结论，如"具有深远意义"、"意义重大"）
- 三段式机械排比与口号化收束
- 模糊主语替代真实来源（如"有研究指出"但无出处、"专家认为"但无具体人名）
- 同义改写堆字数（同一观点用不同说法重复表述）
- 过度使用破折号和连接性短语（如"值得注意的是"、"需要指出的是"）
- 否定式排比（如"不是……而是……不是……而是……"连续使用）
- 超出证据范围的结论

## 建议写法

- 先给可核验事实，再给解释
- 段内保持"主张 → 机制 → 证据 → 小结/过渡"的推进节奏
- 有争议观点注明分歧来源
- 结论只覆盖证据支持的边界
- 使用具体数据、人名、机构名代替模糊表述
- 过渡句基于内容逻辑关系，不使用万能连接词

## 质检清单

在 Step 5 质检阶段，逐项检查：

- [ ] 全文无套话起手段落
- [ ] 无空洞拔高结论（每个结论可回溯到具体证据）
- [ ] 无三段式机械排比
- [ ] 所有"有研究指出"类表述已补充具体来源
- [ ] 无同义改写堆字数现象
- [ ] 破折号使用不超过每千字 2 次
- [ ] 结论边界与证据边界一致
