# 体裁模板

以下模板为可调整的结构骨架。按主题实际需求灵活适配，不要机械照搬。

所有模板须配合标准项目结构使用：在 `05-writing-guide/chapter-outline.md` 中确定选用模板后，在 `06-drafts/` 中按章节落地。

在 Step 0 确定体裁后，只需加载对应的单个文件，无需全部读取。

## 体裁索引

| 体裁 | 文件路径 | 适用场景 |
|------|---------|---------|
| 学术论文 / 硕士论文 | `references/genres/academic_paper.md` | 研究论文、课程论文、学术型报告 |
| 文献综述 | `references/genres/literature_review.md` | 文献梳理、研究现状评述 |
| 行业 / 政策 / 专题研究报告 | `references/genres/research_report.md` | 行业分析、政策研究、机构报告 |
| 个案分析 | `references/genres/case_analysis.md` | 案例研究、典型事件分析 |
| 深度文章 / 特稿 / 评论 | `references/genres/feature_article.md` | 深度报道、杂志长文、述评 |
| 调查 / 解释性报道 | `references/genres/investigative_report.md` | 新闻报道、事实核查 |
| 政策建议 | `references/genres/policy_recommendation.md` | 政策分析中的建议章节 |
