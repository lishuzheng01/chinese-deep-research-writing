# 体裁模板

以下模板为可调整的结构骨架。按主题实际需求灵活适配，不要机械照搬。

所有模板须配合标准项目结构使用：在 `05-writing-guide/chapter-outline.md` 中确定选用模板后，在 `06-drafts/` 中按章节落地。

---

## 学术论文 / 硕士论文

适用于研究论文、课程论文、文献综述、学术型报告。

1. 标题
2. 摘要（研究背景与问题意识、研究对象与范围、材料与方法、核心发现、意义）
3. 关键词
4. 引言 / 问题提出
5. 文献综述 / 理论背景
6. 研究设计与方法说明
7. 主体分析（可分多章）
8. 讨论（意义、局限、争议）
9. 结论
10. 参考文献

写法要求：
- 尽早界定研究对象与范围
- 与已有观点或文献做出关系定位
- 区分材料、方法、发现与诠释
- 使用克制、正式的语气
- 参考 `references/gbt7714_quick_guide.md` 处理引用格式

## 文献综述

1. 研究主题与范围界定
2. 主要学术脉络
3. 代表性观点分类
4. 已有研究的贡献
5. 分歧、盲点与不足
6. 可进入的问题空间

写法要求：
- 按逻辑线索组织，而非按时间简单罗列
- 呈现共识与争议的结构关系
- 指出现有研究的可进入空间

## 行业 / 政策 / 专题研究报告

适用于行业分析、政策研究、机构内部报告、专题研判。

1. 标题
2. 摘要 / 执行摘要
3. 背景与问题定义
4. 现状与关键事实
5. 结构性原因或驱动因素
6. 核心分析
7. 风险与不确定性
8. 结论
9. 建议 / 对策（如适用）
10. 参考资料

写法要求：
- 以决策相关的问题开头
- 事实与建议分层呈现
- 明确标注风险、权衡与执行约束

## 个案分析章节

1. 个案选择理由
2. 背景与关键变量
3. 事件过程 / 制度结构 / 技术路径
4. 机制分析
5. 与其他案例对照
6. 小结

写法要求：
- 说明个案的代表性或特殊性
- 分析因果机制而非仅描述经过

## 深度文章 / 特稿 / 评论

适用于深度文章、专题文、述评、杂志式长文、人物或议题特稿。

1. 标题
2. 导语 / 问题切入
3. 背景铺垫
4. 主体分析或叙事推进
5. 不同观点与张力
6. 深层含义 / 现实影响
7. 结尾收束
8. 资料来源（按需）

写法要求：
- 兼顾节奏感与信息密度
- 可适当使用场景、对比、案例片段
- 保持论证纪律，避免煽情与无据判断

## 调查 / 解释性报道

适用于新闻报道、解释性报道、事实核查式写作。

1. 标题
2. 导语（何人、何事、何时、何地、为何重要）
3. 关键信息与背景
4. 多方说法与归因
5. 数据、文件、现场或历史背景
6. 影响与后续观察点

写法要求：
- 核实日期、人名、机构与事件顺序
- 每项非显而易见的事实陈述须标注来源
- 报道与评论严格分离
- 使用精确、中性的语言

## 政策建议章节

1. 问题定义
2. 政策目标
3. 约束条件
4. 方案比较
5. 建议路径
6. 风险与执行条件

写法要求：
- 建议必须与前文分析形成逻辑闭环
- 列出各方案的优劣与适用边界
